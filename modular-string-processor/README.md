# Modular String Processor

This project is a modular Python program designed to perform
basic string manipulation operations using separate modules.

## Problem Statement
Create a modular Python program to:
1. Convert a string to uppercase  
2. Convert a string to lowercase  
3. Reverse a string  

Each operation is implemented in a **separate Python file**, and a
menu-driven main program integrates all functionalities.

## Project Structure
- `to_uppercase.py` → Converts string to uppercase  
- `to_lowercase.py` → Converts string to lowercase  
- `reverse_string.py` → Reverses the string  
- `main.py` → Menu-driven program integrating all modules  

## Concepts Used
- Modular programming
- String manipulation
- Python functions
- Menu-driven programs
- User input handling
